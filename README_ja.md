## なぜ PicConvert？

ほとんどの画像変換ツールでは、ファイルをリモートサーバーにアップロードする必要があります。PicConvert は Canvas API を使用してブラウザ内ですべてを処理 — データがデバイスから離れることはありません。

| 利点 | 詳細 |
|------|------|
| 🔒 **100% プライベート** | すべての処理はブラウザのメモリで実行。サーバーなし、アップロードなし、トラッキングなし。 |
| ⚡ **即時変換** | 5MB 以下の画像は 500ms 以内に変換。サーバー応答の待ち時間なし。 |
| 🎯 **依存関係ゼロ** | バニラ JavaScript とネイティブ Chrome API で構築。フレームワークなし、不要なものなし。 |
| 🌍 **6 言語対応** | ブラウザの言語を自動検出 — 英語、スペイン語、ドイツ語、日本語、フランス語、中国語。 |
| 📋 **コピーと保存** | 任意の画像を右クリックしてファイルとして保存、またはクリップボードに直接コピー。 |
| 📁 **ドラッグ＆ドロップ** | ローカル画像をポップアップにドロップして素早く変換。 |

## 機能

| 機能 | 説明 |
|------|------|
| 🖼️ **4 形式** | PNG（ロスレス）、JPG（高品質）、WEBP（コンパクト）、AVIF（最高圧縮）に変換 |
| 📋 **クリップボードにコピー** | 右クリック → PNG/JPG/WEBP/AVIF としてコピー — メール、チャット、ドキュメントに直接貼り付け |
| 📊 **ファイルサイズ比較** | トースト通知が元のサイズと変換後のサイズ、節約率を表示 |
| 🎚️ **調整可能な品質** | ポップアップのスライダーで JPG、WEBP、AVIF の書き出し品質を微調整 |
| 🔗 **スマートリンククリーニング** | CDN トラッキングパラメータ（`w=`、`h=`、`format=`、`watermark=` など）を除去してオリジナル画像を取得 |
| 📁 **ドラッグ＆ドロップ** | ローカル画像ファイルをポップアップにドロップして右クリックなしで変換 |
| 🌐 **クロスオリジン対応** | Service Worker 経由で任意のウェブサイトから画像を取得 — CORS と Canvas 汚染の問題を回避 |
| 🔤 **6 言語 i18n** | コンテキストメニューと UI がブラウザの言語に自動一致（EN/ES/DE/JA/FR/ZH-CN） |
| 🛡️ **JPG 白背景塗りつぶし** | PNG → JPG 変換時に透明な背景を自動的に白で塗りつぶし |
| ⏱️ **重複クリックガード** | 2 秒以内の同じ画像への重複クリックを無視 |
| 📦 **大きな画像の処理** | 20MB 超の画像は UI をフリーズさせる代わりに非同期処理インジケータを表示 |

## インストール

### ソースから（デベロッパーモード）

1. ブラウザの拡張機能ページを開く：
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
2. **デベロッパーモード**を有効にする（右上のトグル）
3. **パッケージ化されていない拡張機能を読み込む**をクリックし、`pic-convert` フォルダを選択
4. ✨ PicConvert アイコンがツールバーに表示されます

---

## 使い方

### 右クリック変換

1. ウェブページ上の任意の画像を右クリック
2. コンテキストメニューから **PicConvert** を選択
3. **名前を付けて保存**または**コピー**を選択
4. 形式を選択：PNG、JPG、WEBP、AVIF
5. 完了 — ファイルが即座にダウンロードされるか、画像がクリップボードにコピーされます

### ドラッグ＆ドロップ

1. ツールバーの PicConvert アイコンをクリックしてポップアップを開く
2. ローカル画像ファイルをドロップゾーンにドラッグ
3. 対象形式を選択
4. 変換されたファイルが自動的にダウンロードされます

### 品質設定

ポップアップを開いてスライダーで JPG、WEBP、AVIF の書き出し品質を調整します。PNG は常にロスレスです。設定は自動的に保存されます。

## プライバシー

PicConvert はプライバシーを基本原則として構築されています：

- ✅ **データアップロードゼロ** — すべての画像処理はブラウザのメモリで実行
- ✅ **分析なし** — トラッキングなし、テレメトリなし、リモート呼び出しなし
- ✅ **Cookie なし** — ブラウザ Cookie の読み書きなし
- ✅ **閲覧履歴なし** — ブラウザデータへのアクセスなし
- ✅ **一時メモリのみ** — 画像は変換中にメモリに存在し、完了後すぐに破棄
- ✅ **最小限の権限** — 厳密に必要なもののみ要求：`contextMenus`、`downloads`、`offscreen`、`storage`、`clipboardWrite`

## 仕組み

```
画像を右クリック
       ↓
Service Worker が画像 Blob を取得（CORS を回避）
       ↓
Blob をオフスクリーンドキュメントに送信（Canvas アクセス可能な隠し DOM）
       ↓
Canvas がネイティブ解像度で画像を描画
       ↓
指定された品質で対象形式として書き出し
       ↓
データ URL を返す → ダウンロードまたはクリップボードコピーを実行
       ↓
すべての一時リソースを破棄（Blob、Canvas、画像要素）
```

> **なぜオフスクリーン？** Chrome の Manifest V3 はバックグラウンドを Service Worker として実行し、DOM アクセスがありません。Canvas API には DOM が必要なため、Chrome のオフスクリーン API を使用して画像処理用の隠しドキュメントを作成します。

## 書き出し品質のデフォルト

| 形式 | 品質 | 備考 |
|------|------|------|
| PNG | ロスレス | 品質設定なし — 常に完全な品質と透明度を保持 |
| JPG | 92 | 高品質、サイズと鮮明度の良好なバランス |
| WEBP | 90 | 最新形式、同等品質で JPG より約 25-35% 小さい |
| AVIF | 70 | 次世代形式、WEBP より約 20-30% 小さい、Web 配信に最適 |

すべての品質値はポップアップのスライダーで調整可能（範囲：10–100）。
